TETRIS

React | React-Hooks | CSS

A fully-functioning Tetris game built for the web. User could play the game from their browser. The controls are simply the arrow key on the keyboard. The game allows the user to rotate the pieces, simply by pressing the up-key on the keyboard. Note that this app does not support mobile-compatibility yet.

Tetris is being hosted here: Tetris

Repository can be found here: Github

Sudoku Solver

HTML | CSS | Typescript

A web application that allows the user to submit a sudoku board and the application will then find the sudoku solution. It has the option to "auto-fill" that will create a sample sudoku board and "clear" to allow the user to clear the sudoku question board. The solution is written using Generative Recursion and backtracking search.

Sudoku Solver is being hosted here: Sudoku Solver

Repository can be found here: Github

Natours

HTML | CSS | SASS

Natours is a fully responsive travelling-tour website where user could check various tours, learn the perks of joining these tours and read some of the awesome experiences from other users that have tried these tours. Natours was created as part of Jonas Schmedmedtmann's course from Udemy called the "Advanced CSS and SASS Course".

Natours is being hosted here: Natours

Repository can be found here: Github

BURGER MAKER

About Me

My name is Michael Suriawan; I am a Mechanical Engineering student with a deep passion towards Web Development. I have been coding for almost three years. After building several dynamic projects, I realized that I am enjoying it even more and more today. The experience of designing a web component that could change how the user interact with a webpage has always been exhilarating for me.

I have designed and built several notable projects with React.js ,Redux, Node.js and vanilla JavaScript. I have also created two back-end projects which include a company blog and a fake "Reddit" website with the help of both Python3 and Django. My other notable skills include C, Java & Matlab and I could not wait to learn more!

I have completed a few Data Structure and Algorithm courses to help me become a more efficient programmer. I believe that I will be a great addition to any team and would aid in designing the best inventions out there that I can. All I need is an opportunity for me to do exactly that. Currently, I am looking for a technical-internship position starting in January 2020.
